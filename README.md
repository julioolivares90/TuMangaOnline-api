# TuMangaOnline-api 

## **:package: Principales herramientas usadas**

- [x] fiber
- [x] go colly


## :rocket:  tumangaonline-api API enlace
https://tumangaonline-api.herokuapp.com/

### iniciar el servidor usando go

```
go run App.go
```
